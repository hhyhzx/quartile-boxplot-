一、数据的集中趋势

描述数据"中心在哪里"：平均数（数据的平衡点）、中位数（排序后正中间的数值）、众数（出现次数最多的值）、加权平均数（带权重的平均）。

动画：平均数 — 数据的"平衡点" GIF

所有数据之和除以个数。把每个数据到平均数的正负离差加起来 = 0，像天平一样平衡。

三种度量对比 图

同一组数据，三种度量结果不同。选哪个取决于数据特征和分析目的。

加权平均数 图

期末总评 = 各项分数 × 对应权重。权重反映重要程度。

平均数 vs 中位数：谁更"稳健"？ 图

当数据中有极端值时（如某同学考了25分），平均数被大幅拉低，而中位数几乎不受影响。这就是"稳健性"。

二、数据的离散程度

描述数据"散得有多开"：离差（xi−x̄）、离差平方和 Q（避免正负抵消）、方差 s²（Q÷n）、标准差 s（√方差，单位与原始数据一致）。

动画：离差 → 离差平方和 → 方差 GIF

红色线 = 负离差（低于平均数），绿色线 = 正离差（高于平均数）。离差平方和 Q = Σ(xi−x̄)²，方差 s² = Q / n。

完整计算流程 图

五步法：数据 → 离差 → 平方 → 求和(Q) → 除以n(方差) → 开方(标准差)。

两组数据对比 图

两班平均分相同(80)，但A班方差小（整齐），B班方差大（参差不齐）。方差区分了离散程度。

三、四分位数与箱线图 ★ 新课标新增

四分位数：将排序后的数据四等分的三个点 Q1 (25%)、Q2 (50%=中位数)、Q3 (75%)。箱线图：用五数概括（min, Q1, Q2, Q3, max）绘制的图形。IQR = Q3−Q1，反映中间50%数据的跨度。

动画：四分位数标注 + 箱线图构建 GIF

上半：数据分区（前25% / 25-50% / 50-75% / 后25%）。下半：箱线图四步构建（端点→箱体→中位线→须线）。

四分位数计算详解 图

n=15（奇数）：Q2 取第8个，Q1 取前7个的中位数，Q3 取后7个的中位数。

多组数据对比箱线图 图

四个班级的箱线图并列展示。箱体=Q1~Q3，粗线=中位数，菱形=平均数，须=min~max。

四、数据的分组

将数据分段整理：组距（每段的宽度）、频数（每段有几个数据）、频率（频数÷总数）、直方图（图形呈现）。分组原则：组内差异小，组间差异大。

动画：原始散点 → 分组 → 频数统计 → 直方图 GIF

50人成绩：先确定组距(10分)，再统计每组人数，最后画出直方图。组内差异小，组间差异大。

频数分布表 + 直方图 图

频数分布表包含：组别、频数、频率、累计频数。红色折线为频数多边形。

组距对比：过大 vs 过小 vs 合适 图

组距过大→信息丢失；组距过小→太琐碎；组距合适→看清分布形态。

全章知识结构总览

模块核心概念关键公式 / 方法
集中趋势 平均数、加权平均数、中位数、众数 x̄ = Σxi / n；加权 x̄ = Σ(wi·xi) / Σwi
离散程度 离差、离差平方和、方差、标准差 Q = Σ(xi−x̄)²；s² = Q/n；s = √(s²)
分布特征 ★新 四分位数(Q1/Q2/Q3)、箱线图、IQR、百分位数 IQR = Q3−Q1；五数概括(min,Q1,Q2,Q3,max)
数据分组 组距、频数、频率、直方图、分组原则 组距 = 极差/组数；频率 = 频数/n
