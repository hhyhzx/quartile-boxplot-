一、数据的集中趋势

描述数据"中心在哪里"：平均数（数据的平衡点）、中位数（排序后正中间的数值）、众数（出现次数最多的值）、加权平均数（带权重的平均）。

视频讲解：平均数 — 数据的"平衡点" MP4 语音

▶ 语音讲解 🔍 点击浮屏
所有数据之和除以个数。把每个数据到平均数的正负离差加起来 = 0，像天平一样平衡。含中文语音讲解。

加权平均数 图

🔍
期末总评 = 各项分数 × 对应权重。权重反映重要程度。

平均数 vs 中位数：谁更"稳健"？ 图

🔍 点击放大
当数据中有极端值时（如某同学考了25分），平均数被大幅拉低，而中位数几乎不受影响。这就是"稳健性"。

二、数据的离散程度

描述数据"散得有多开"：离差（xi-x̄）、离差平方和 Q（避免正负抵消）、方差 s²（Q/n）、标准差 s（方差的算术平方根）。

视频讲解：离差 -> 离差平方和 -> 方差 MP4 语音

▶ 语音讲解 🔍 点击浮屏
红色线 = 负离差，绿色线 = 正离差。离差平方和 Q = Σ(xi-x̄)²，方差 s² = Q / n。含中文语音讲解。

完整计算流程 图

🔍
五步法：数据 -> 离差 -> 平方 -> 求和(Q) -> 除以n(方差) -> 开方(标准差)。

🔍

三、四分位数与箱线图 * 新课标新增

四分位数：将排序后的数据四等分的三个点 Q1 (25%)、Q2 (50%=中位数)、Q3 (75%)。箱线图：用五数概括（min, Q1, Q2, Q3, max）绘制的图形。IQR = Q3-Q1。

四、数据的分组

将数据分段整理：组距（每段的宽度）、频数（每段有几个数据）、频率（频数/总数）、直方图（图形呈现）。分组原则：组内差异小，组间差异大。

全章知识结构总览

模块核心概念关键公式 / 方法
集中趋势 平均数、加权平均数、中位数、众数 x̄ = Σxi / n；加权 x̄ = Σ(wi xi) / Σwi
离散程度 离差、离差平方和、方差、标准差 Q = Σ(xi-x̄)²；s² = Q/n；s = sqrt(s²)
分布特征 *新 四分位数(Q1/Q2/Q3)、箱线图、IQR、百分位数 IQR = Q3-Q1；五数概括(min,Q1,Q2,Q3,max)
